# Micro-PMU Based Smart Distribution Monitoring System

This project focuses on real-time monitoring of electrical distribution systems using IoT-based data acquisition and web-based visualization.

## Features

* Real-time monitoring of electrical parameters
* Fault detection (<1 sec response)
* Web-based dashboard for visualization and analysis
* Energy monitoring and insights

## Applications

* Suitable for industrial, commercial, and residential power monitoring
* Can be deployed in industries, colleges, hospitals, and smart homes
* Supports integrated IoT-based hardware and software monitoring systems

## Tech Stack

JavaScript • HTML • C • Embedded Systems • IoT
